README - Compilation Instructions:
1. Extract src.zip
2. Navigate to the folder in your terminal.
3. Type "make play" to launch the game.

4. Play the game! Follow onscreen instructions to play 2-player or vs ai.
Try using space-to-space notation "e2 e4" to move from e2 to e4 or algebraic
notation "Nf3" to move a knight to f3. You can also give pawn moves with just
the end square, for example "e4".
